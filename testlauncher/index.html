Welcome to Eaglercraft Launcher

This site was created and developed by missionoperation. The goal of this launcher is to provide a clean, in-browser way to play different versions of Eaglercraft, explore custom clients, and try out fun extras — all without leaving the site.

What’s on this site?

  • Normal Versions — Play classic eras like 1.3, 1.5.2, 1.7.3, 1.8.8, 1.9, and 1.12.2, plus experimental builds like Indev.
  • Clients — Launch polished, mod-friendly clients such as Astra, Kone, Pi, Resent, Shadow, and Solar.
  • Hacked Clients — Test experimental/sandbox clients like DragonX V4, Nebula, OddFuture, UwU, and fuchsiax.
  • Extras — Try alternative takes like Paper Minecraft (2D), Mobile builds, Minecraft 3D, and Another Minecraft.

Each game launches directly in-site. Use ESC and hover at the top edge of the screen to access session controls at any time.
